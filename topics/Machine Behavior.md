# Machine Behavior

Studying AI systems the way we study animals — observationally, behaviorally, ecologically. Coined by Iyad Rahwan et al. in a 2019 Nature paper.

## Key Texts
- [[Iyad Rahwan]] et al., "Machine Behaviour" (Nature, 2019)
- Amber's own work on agent ethology and machine behavior
