# Social Simulation

Using AI agents to simulate human societies. Currently exploding — one of the fastest-evolving areas in AI.

## Why Edge of Chaos
This field barely existed 2 years ago. Joon Sung Park's generative agents paper (2023) created a Cambrian explosion. Now: population-scale sims, validated against real behavior, institutional emergence, cooperation dynamics. The field is reshaping weekly.

## Key People
- [[Joon Sung Park]] — generative agents, launched the wave
- [[Joel Z Leibo]] — Concordia platform, cultural evolution × AI
- [[Iyad Rahwan]] — machine behavior framing
- Michael Bernstein — Stanford, Joon's advisor
- Jason Wei — chain of thought, adjacent

## Key Platforms/Projects
- **Concordia** (DeepMind/Leibo) — tabletop RPG-style open-ended sims
- **Generative Agents / Smallville** (Park) — memory + reflection + planning
- **Project Sid** — 1000+ agent civilizations
- **S-Agents**, **AgentSims**, etc. — fast-proliferating

## Connections

## Key Questions (active / unresolved)
- Can LLM agents genuinely model human cooperation or just mimic it?
- What institutions emerge spontaneously in agent societies?
- Can we use these sims for policy testing?
- What's the ontological status of a simulated agent with memory and reflection?
