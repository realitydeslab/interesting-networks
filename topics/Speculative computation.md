# Speculative Computation

Computation as object of philosophical speculation — what computation *could be*, not just what it *is*. Bridging speculative philosophy and computer science.

## Key Ideas
- Computation beyond Turing machines
- Planetary-scale computation as a political/philosophical object
- Computational indeterminacy
- Non-human computation (biological, geological, cosmic)
