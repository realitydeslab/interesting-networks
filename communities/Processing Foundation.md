# Processing Foundation

## ⚡ Recent Updates
- **2026-02-26:** Created stub note. Discovered through **[[Lauren Lee McCarthy]]** research as board member.

Foundation supporting creative coding education and open-source software development. Stewards the Processing ecosystem including p5.js, Processing.py, and related educational tools used by millions of artists, designers, and educators worldwide.

## Background
Foundation formed to support the Processing software and community. Manages open-source creative coding tools and educational initiatives making programming accessible to artists and designers.

## Key Projects
- **Processing** — Original Java-based creative coding environment
- **p5.js** — JavaScript creative coding framework (led by **[[Lauren Lee McCarthy]]**)
- **Processing.py** — Python version
- **Educational initiatives** — Workshops, documentation, community building

## Leadership & Community

## Key Venues & Events

## Why Interesting
**Infrastructure for creative coding education:** Provides tools used by millions to learn programming for creative purposes. Bridge between computer science education and arts education.

**Open-source sustainability model:** Long-term stewardship of creative tools. Model for maintaining artistic/educational software infrastructure.

**Global arts education influence:** Processing/p5.js taught in art schools worldwide. Shapes how next generation learns programming for creative practice.
