# Gnosis Guild

Small team that established the Zodiac open standard for composable, modular software for organizations. Core contributor to decentralized governance infrastructure.

## Mission & Objectives
Develops composable governance tools and pattern languages for organizations. Focuses on modular organizational software that enables flexible, customizable governance systems.

## Key People
- [[Kei Kreutler]] — Co-creator/Co-founder
- Team of governance and protocol developers

## Key Contributions
- **Zodiac Protocol** — Open standard for composable governance tools. Over $3 billion in transactions processed.
- Used by major DAOs including ENS DAO, Balancer
- Modular tools including Roles Modifier, Exit Module, Delay Modifier, Reality Module, Bridge Module

## Key Publications
- [Composable Expansion Packs for Decentralized Organizations](https://law.mit.edu/pub/composableexpansionpacks/release/1) — MIT Computational Law Review

## Why Interesting
Practical implementation of composable governance systems. Not just theoretical work but real infrastructure processing billions in transactions. Model for how to build technical tools with philosophical grounding in organizational design.
