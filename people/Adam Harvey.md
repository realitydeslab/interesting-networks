# Adam Harvey

## ⚡ Recent Updates
- **2026-02-26:** Created stub note. Discovered through [[Kyle McDonald]] research — shared SXSW 2014 panel on Hacking Surveillance Culture (with Heather Dewey-Hagborg, Kyle McDonald, Colin Nagy).

Artist and researcher known for **CV Dazzle** — camouflage designs that defeat facial recognition software using high-contrast geometric patterns. Pioneer in anti-surveillance aesthetics. Also created **HyperFace** (a pattern that overwhelms face detection by presenting too many false faces) and **MegaPixels** (investigating datasets used to train facial recognition). Based Berlin.

## Key Works
- **[CV Dazzle](https://cvdazzle.com/)** — anti-surveillance makeup/fashion designs that defeat face detection
- **[HyperFace](https://ahprojects.com/hyperface/)** — countermeasure camouflage
- **[MegaPixels](https://megapixels.cc/)** — investigates face recognition training datasets
- **[VFRAME](https://vframe.io/)** — visual media analysis tools for human rights documentation

## Why Interesting
Central figure in critical computer vision and anti-surveillance aesthetics. Rigorous technical understanding + artistic/political stance. Direct intellectual neighbor of [[Kyle McDonald]] and [[Heather Dewey-Hagborg]]. His methodology — using deep technical knowledge to subvert technical systems — is a model for critical computing through practice.
