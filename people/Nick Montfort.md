# Nick Montfort

MIT media theorist, computational poet, and co-founder of [[Platform Studies]] with Ian Bogost. Co-author of *10 PRINT CHR$(205.5+RND(1)); : GOTO 10* (MIT Press, 2012) — a collaborative close reading of a single line of BASIC code that became a landmark in critical code studies. Works at the intersection of electronic literature, platform studies, and computational creativity. Key figure in Software Studies ecosystem.
