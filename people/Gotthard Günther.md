# Gotthard Günther (1900–1984)

German-American philosopher and logician. BCL (Biological Computer Laboratory) regular professor (since 1967). Developed **polycontexturality** — a non-Aristotelian logic framework for handling multiple observer positions simultaneously. His core insight: classical two-valued logic (true/false) is insufficient for modeling self-referential and reflexive systems that include observers. Polycontexturality proposes that reality has multiple, irreducibly different logical contexts ("textures"), none of which can subsume the others.

## Key Works
- *Idee und Grundriss einer nicht-Aristotelischen Logik* (1959) — foundational statement of polycontexturality
- *Beiträge zur Grundlegung einer operationsfähigen Dialektik* (3 vols., 1976–1980) — comprehensive theory
- Essays collected in *Beiträge* and in the Cybernetics journal *Kybernetes*

## Why Interesting
Günther's polycontexturality is a formal framework for handling the multiple-observer problem that second-order cybernetics identified but struggled to formalize. If agent ethology involves multiple observers (researcher, AI, environment, users) each operating within different "logical textures," Günther's framework provides vocabulary. Underread and ahead of its time — potentially useful for thinking about multi-agent systems where different agents have different observational standpoints.
