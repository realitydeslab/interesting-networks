# Mark Marino

USC writing professor and founder of Critical Code Studies (CCS) — the approach that applies literary hermeneutics to source code, reading it as culturally significant text rather than merely functional instruction. Author of *Critical Code Studies* (MIT Press, 2020). Organizes the Critical Code Studies Working Group. Key figure at the intersection of Software Studies and digital humanities.
